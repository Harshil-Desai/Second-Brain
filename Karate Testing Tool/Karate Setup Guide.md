# Karate Setup Guide

## Step 1 — Install Java

1. Download Java: https://download.oracle.com/java/25/latest/jdk-25_windows-x64_bin.exe
2. Run the installer (choose defaults).
3. Verify the installation — open Command Prompt and run:

   ```
   java -version
   ```

## Step 2 — Get the Karate JAR

Download the JAR: https://github.com/karatelabs/karate/releases/download/v1.5.2/karate-1.5.2.jar
